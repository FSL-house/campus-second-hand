# 商品图片目录

你可以把本地静态商品图片放到这个目录，例如：

- `keyboard.jpg`
- `book.png`
- `lamp.jpeg`

如果图片放在：

- `public/images/keyboard.jpg`

那么发布商品时，图片地址填写：

- `/images/keyboard.jpg`

注意：

1. 浏览器访问时不需要写 `public`
2. 图片文件名尽量使用英文，避免空格和中文
3. 修改或新增图片后，前端页面刷新即可看到效果

项目已经自带了几个可直接使用的示例图片：
- `/images/java-book.svg`
- `/images/power-bank.svg`
- `/images/lamp.svg`
- `/images/demo.svg`
